  https://github.com/HuskyLens/HUSKYLENSUploader
**ขั้นตอนการติดตั้ง**
 - เลือกไฟล์ zip ที่ชื่อว่า  **HUSKYLENS Uploader-V2.1.zip**
 - จากนั้นจะได้ไฟล์ที่ชื่อว่า  **HUSKYLENS Uploader-V2.1.exe** 
    ให้ทำการดาวห์โหลดเพื่อนำไปเปิดบน window
